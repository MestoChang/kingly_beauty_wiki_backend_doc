# 如何新增關鍵字

1. 進入關鍵字列表

    ![關鍵字列表](img/keyword-list-01.png)

2. 點擊右上，新增關鍵字

    ![關鍵字列表](img/keyword-list-02.png)

3. 出現填寫關鍵字欄位

    ![新增關鍵字欄位](img/add-keyword.png)

4. 點擊 `送出`，系統檢查沒有重複關鍵字即新增成功

    ![新增關鍵字成功](img/add-keyword-success.png)

    :::warning
    如果同時新增多個關鍵字，只要有一個關鍵字重複即無法送出！
    :::
